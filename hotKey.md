### 文件操作
```
ctrl + /                注释  
 
ctrl + shift + /        注释

ctrl +ｄ                复制行

ctrl + g                查找行

ctrl + shift + up/down  代码向上/向下

ctrl + b / ctrl + click 快速打开该方法或类

ctrl + alt + b          跳到该方法实现处

ctrl + shift + i        显示当前方法详细信息

alt + up/down           方法之间跳转

alt + shift + f         添加到收藏夹
 
ctrl + shift + alt + n  通过word全局查找

ctrl + alt + t          用if 或 其他 包裹选中代码

ctrl +　shift +　u       选中英文大小写切换

ctrl + delete           删除至文字结束

ctrl + backSpace        删除至文字开头

F11                     添加标签

shift + enter           跳到下一行
```
###文档操作
